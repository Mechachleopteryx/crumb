AGGREGATING

AGGREGATING

This extension is aggregating, meaning that it may generate outputs based on several annotated input classes and it respects the constraints imposed on aggregating processors. It is common for Crumb producer extensions to be aggregating and unusual for consumer extensions to be aggregating.
