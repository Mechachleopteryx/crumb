producerIncrementalType

open fun producerIncrementalType(processingEnvironment: ProcessingEnvironment): CrumbExtension.IncrementalExtensionType

Determines the incremental type of this Extension.

The ProcessingEnvironment can be used, among other things, to obtain the processor options, using ProcessingEnvironment.getOptions.

The actual incremental type of the Crumb processor as a whole will be the loosest incremental types of the Extensions present in the annotation processor path. The default returned value is IncrementalExtensionType.UNKNOWN, which will disable incremental annotation processing entirely.
