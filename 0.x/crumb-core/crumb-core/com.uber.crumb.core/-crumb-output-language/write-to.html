writeTo

abstract fun writeTo(filer: Filer, packageName: String, fileName: String, originatingElements: Set<Element> = emptySet()): BufferedSink

Writes crumb metadata for this language.

Parameters

filer - A Filer to write with.

packageName - The package to write to. Note that this should be the package that all metadata index-holder types are written to, and not necessarily the package name of the source element.

fileName - The file name.

originatingElements - Any originating elements for the metadata.

Return
A BufferedSink to write metadata to. This will (only) be written to the eventual CrumbIndex once BufferedSink.close is called.
