CrumbLog(prefix: String, debugEnabled: Boolean = false, client: CrumbLog.Client = DefaultClient)

Tiny logger implementation for debugging CrumbManager, adapted from the Android DataBinding library.

Note that this is just for logging of Crumb internals, and not intended for general use. Thus, all the actual logging functions are internal visibility only.

Parameters

prefix - The prefix to use in logs from this logger.

debugEnabled - Whether debug logs are enabled. Default is false.

client - Optional custom Client for handling logs. Default is DefaultClient, which just prints to stdout and stderr.
