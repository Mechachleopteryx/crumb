store

fun store(packageName: String, fileName: String, outputLanguage: CrumbOutputLanguage, originatingElements: Set<Element> = emptySet()): BufferedSink

This facilitates writing data to a CrumbIndex type in the given packageName.fileName with the contents written to the returned BufferedSink.

Parameters

packageName - The package name to use for the file in writing. Note that this should be the package that all metadata index-holder types are written to, and not necessarily the package name of the source element.

fileName - The file name to use in writing.

outputLanguage - The target output language.

originatingElements - Any originating elements for the metadata.

Return
A BufferedSink to write metadata to. This will (only) be written to the eventual CrumbIndex once BufferedSink.close is called.
